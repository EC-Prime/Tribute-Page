# Walter White Tribute Page

A tribute page dedicated to Walter White, the fictional protagonist of *Breaking Bad* — built as part of the [freeCodeCamp Responsive Web Design Certification](https://www.freecodecamp.org/learn/2022/responsive-web-design/).

## 🔗 Live Demo
https://heisenburgbrb.netlify.app

---

## 📌 About

This project is a tribute page for Walter White, tracing his transformation from a high school chemistry teacher to one of the most iconic antiheroes in television history. The page includes a biographical timeline, a curated quote, and a link to his Wikipedia entry.

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, animations, transitions)

---

## ✨ Features

- Grayscale image filter that reveals full color on hover
- Fade-in animations on page load
- Smooth image zoom on hover
- Fully responsive layout
- Semantic HTML structure (`figure`, `figcaption`, `section`, `blockquote`)
- Custom list styling

---

## 📚 What I Learned

- Structuring a page with semantic HTML elements
- Using CSS `filter` and `transition` for visual effects
- Centering elements with `margin: auto`
- Building responsive images with `max-width` and `height: auto`
- CSS `@keyframes` animations for entrance effects
